📝 Sobre o Projeto

Este repositório é o meu laboratório de estudos em Java. O foco principal é a transição da lógica de programação estruturada para o paradigma de Orientação a Objetos (OO).

Ao longo dos exercícios, aplico conceitos fundamentais para criar códigos mais modulares, reutilizáveis e organizados:

    Lógica de Programação: Domínio de algoritmos, estruturas de repetição, condicionais e manipulação de fluxos de dados.

    Abstração e Encapsulamento: Criação de classes que representam entidades do mundo real com estados (atributos) e comportamentos (métodos) bem definidos.

    Herança e Polimorfismo: Implementação de estruturas que permitem a especialização de classes e o tratamento genérico de objetos.

    Collections API: Uso estratégico de List, Set e Map para gerenciamento eficiente de dados.

Cada exercício aqui presente reflete um degrau na minha evolução técnica, buscando sempre aplicar as melhores práticas da linguagem.
Por que essa descrição é boa?

    Palavras-Chave: Ela cita termos técnicos (Herança, Polimorfismo, Collections) que são muito bem vistos por quem avalia código.

    Narrativa: Mostra que você tem consciência do que está estudando, e não está apenas copiando código.

    Escalabilidade: Serve tanto para exercícios simples quanto para pequenos sistemas que você venha a criar.
